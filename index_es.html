Entorno de desarrollo para Arduino

Este entorno de desarrollo permite a los estudiantes acceder a una herramienta a tiempo real para desarrollar, compilar y ejecutar código en lenguaje C para Arduino.

Introduce tu código

Tu sesión durará 10 minutos. Puedes usar el editor de código para crear un nuevo Sketch, cargarlo desde tu equipo o utilizar uno de los ejemplos predefinidos disponibles.

Verifica tu código

Una vez que hayas terminado el Sketch, compila tu código para comprobar que la sintaxis es correcta y puede ser ejecutado.

Ejecuta el Sketch

Cuando el Sketch haya sido compilado correctamente, puedes ejecutarlo y ver el fruto de tu trabajo a tiempo real a través de la webcam.
